U ovoj godini mogao sam pročitati:

32.3 knjige do sada.

Matematika

T (vrijeme čitanja po danu) = 30 [min / dan]
V (prosječna brzina čitanja) = 200 [riječi / dan]
Srps (prosječni broj riječi po stranici) = 300 [riječi / stranica]
Sspk (prosječni broj stranica po knjizi) = 300 [stranica / knjiga]
----------------------------------------------------------------
S (broj knjiga po danu) = ? [knjiga / dan]

Srpd (pročitani broj riječi po danu) = T * V = 30 [min / dan] * 200 [riječi / dan] = 6000 [riječi / dan]
Sspd (pročitani broj stranica po danu) = Srpd / Srps = 6000 [riječi / dan] / 300 [riječi / stranica] = 20 [stranica / dan]
S = Sspd / Sspk = 20 [stranica / dan] / 300 [stranica / knjiga] = 1 / 15 [knjiga / dan]
